## 2. Fluxo de Registro de Marca (13 Etapas)

```
[INÍCIO JORNADA MARCA]
    │
    ▼
[ETAPA 1] O que é marca
    │ Conteúdo educativo + exemplos
    │ Botão: "Entendi, próximo passo"
    ▼
[ETAPA 2] O que pode e o que não pode ser marca
    │ Lista de regras + exemplos visuais
    │ Alerta: "Sua ideia se encaixa?"
    ▼
[ETAPA 3] Busca prévia de marca
    │ ┌─ Buscar no sistema (se integração ativa)
    │ └─ Abrir link oficial do INPI
    │ Resultado: encontrou similar? → Alerta
    ▼
[ETAPA 4] Escolher Classe Nice
    │ Explicação simples + buscador de classes
    │ Sugestões baseadas no tipo de negócio
    ▼
[ETAPA 5] Preparar documentos
    │ Checklist dinâmico:
    │   PF: CPF, RG, comprovante
    │   PJ: CNPJ, contrato social
    │   MEI: CCMEI
    │ + logotipo em alta resolução (se marca mista/figurativa)
    ▼
[ETAPA 6] Cadastro no e-INPI
    │ Link direto → https://www.gov.br/inpi
    │ Passo a passo com screenshots
    │ "Já tenho cadastro" → pular
    ▼
[ETAPA 7] Emissão da GRU
    │ Link direto → sistema de GRU
    │ Explicação do valor (PF, PJ, MEI)
    │ Alerta: "Guarde o número da GRU"
    ▼
[ETAPA 8] Pagamento da GRU
    │ Instruções de pagamento
    │ Prazos
    │ "Já paguei" → próximo
    ▼
[ETAPA 9] Protocolo no e-Marcas
    │ Link direto → e-Marcas
    │ Passo a passo detalhado
    │ Campos a preencher
    │ Dicas para não errar
    ▼
[ETAPA 10] Acompanhamento
    │ Link → Revista da PI
    │ Como buscar seu processo
    │ Prazos típicos
    │ Configurar lembrete
    ▼
[ETAPA 11] Oposição e Exigência
    │ O que é cada uma
    │ Prazos de resposta
    │ Quando buscar ajuda profissional
    ▼
[ETAPA 12] Concessão
    │ O que acontece quando aprovam
    │ Certificado
    │ Validade (10 anos)
    ▼
[ETAPA 13] Renovação
    │ Quando renovar
    │ Como renovar
    │ Alertas de prazo
    │
    ▼
[JORNADA CONCLUÍDA] 🎉
    │ Resumo final
    │ PDF
    │ Próximos cuidados
```

## 5. Fluxo da Triagem Inteligente

```
[INÍCIO TRIAGEM]
    │
    ▼
[P1] "O que você quer proteger?"
    ├── "Um nome, logotipo ou símbolo" ───────► score_marca +3
    ├── "Uma invenção ou processo novo" ──────► score_patente +3
    ├── "O visual/aparência de um produto" ──► score_di +3
    └── "Não tenho certeza" ─────────────────► continua sem pontuação
    │
    ▼
[P2] "O que melhor descreve o que você criou?"
    ├── "Um nome ou imagem que identifica meu negócio" ► score_marca +2
    ├── "Algo que funciona de um jeito novo" ──────────► score_patente +2
    ├── "Uma melhoria prática em algo que já existe" ──► score_patente +1, score_mu +1
    ├── "Um formato visual ou design exclusivo" ───────► score_di +2
    └── "Mais de um dos itens acima" ──────────────────► pedir detalhes
    │
    ▼
[P3] "Sobre a parte principal da sua proteção:"
    ├── "É mais sobre como as pessoas reconhecem meu produto/serviço" ► score_marca +2
    ├── "É mais sobre como meu produto funciona tecnicamente" ────────► score_patente +2
    ├── "É mais sobre como meu produto se parece visualmente" ────────► score_di +2
    │
    ▼
[P4] "Você já pesquisou se existe algo parecido registrado?"
    ├── "Sim, pesquisei" ───────► registra, continua
    ├── "Não, ainda não" ───────► sugestão de busca, continua
    │
    ▼
[P5] "Você é:"
    ├── "Pessoa Física" ────► registra perfil PF
    ├── "Pessoa Jurídica" ──► registra perfil PJ
    ├── "MEI" ──────────────► registra perfil MEI
    │
    ▼
[P6] "Seu produto/marca já está em uso no mercado?"
    ├── "Sim, já uso" ──────► registra, se patente: alerta sigilo
    ├── "Ainda não" ────────► registra
    ├── "Em desenvolvimento" ► registra
    │
    ▼
[CÁLCULO]
    │
    ├── score_marca >= 5 ──────────► RESULTADO: MARCA (confiança alta)
    ├── score_patente >= 5 ────────► RESULTADO: PATENTE (confiança alta)
    ├── score_di >= 5 ─────────────► RESULTADO: DESENHO INDUSTRIAL (confiança alta)
    ├── melhor score 3-4 ──────────► RESULTADO com confiança média + aviso
    ├── empate ou score baixo ─────► INCONCLUSIVO → sugere especialista
    │
    ▼
[TELA DE RESULTADO]
    ├── Tipo identificado + explicação
    ├── Confiança (alta/média/baixa)
    ├── CTA: "Começar passo a passo"
    ├── CTA: "Quero falar com especialista"
    └── Link: "Refazer triagem"
```
