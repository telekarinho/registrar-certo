# Checklist Jurídico e Editorial — Registrar Certo

## 1. Compliance Legal

### LGPD (Lei Geral de Proteção de Dados)
- [ ] Política de Privacidade completa e acessível
- [ ] Termos de Uso completos e acessíveis
- [ ] Banner de cookies com opção de aceitar/recusar
- [ ] Consentimento explícito no cadastro (checkbox não pré-marcado)
- [ ] Registro de data/hora do consentimento
- [ ] Possibilidade de revogação do consentimento
- [ ] Direito de acesso aos dados pessoais
- [ ] Direito de exclusão (soft delete com período de retenção)
- [ ] Minimização de dados: coletar apenas o necessário
- [ ] Base legal definida para cada tratamento de dados
- [ ] Encarregado de dados (DPO) designado ou previsto
- [ ] Relatório de impacto (RIPD) previsto
- [ ] Compartilhamento com terceiros documentado (OpenAI, SMTP, etc.)
- [ ] Prazo de retenção de dados definido
- [ ] Logs de auditoria de acesso a dados pessoais

### Marco Civil da Internet
- [ ] Registro de logs de acesso (6 meses mínimo)
- [ ] Termos de uso claros sobre responsabilidade do conteúdo
- [ ] Canal para reclamações e denúncias

### Código de Defesa do Consumidor
- [ ] Informações claras sobre serviços pagos
- [ ] Direito de arrependimento (7 dias para compras online)
- [ ] Preços transparentes
- [ ] Canal de atendimento acessível

## 2. Disclaimers Obrigatórios

### Disclaimer Principal (rodapé + páginas de fluxo)
> "O Registrar Certo é um guia educativo e operacional. Não somos escritório de advocacia, não prestamos assessoria jurídica e não representamos o INPI. O pedido oficial de registro é feito exclusivamente nos sistemas oficiais do INPI (gov.br/inpi)."

### Disclaimer de Triagem
> "O resultado desta triagem é uma orientação inicial baseada nas suas respostas. Para casos complexos, recomendamos consultar um profissional especializado em propriedade industrial."

### Disclaimer de Busca
> "A busca realizada neste sistema é uma consulta auxiliar e não substitui a busca oficial do INPI. Os dados podem estar desatualizados ou incompletos."

### Disclaimer de IA
> "As orientações geradas por inteligência artificial são de natureza educativa. Não constituem parecer jurídico. Revise sempre as informações nos canais oficiais."

### Disclaimer de Checklist/PDF
> "Este documento é um guia de apoio. Confirme sempre os requisitos atualizados no site oficial do INPI antes de realizar qualquer procedimento."

### Disclaimer de Custos
> "Os valores mencionados são referências baseadas nas tabelas vigentes do INPI e podem sofrer alteração. Consulte sempre os valores atualizados no site oficial."

## 5. Checklist Editorial

### Tom e Linguagem
- [ ] Todo conteúdo em português do Brasil
- [ ] Linguagem simples e direta
- [ ] Sem juridiquês desnecessário
- [ ] Tom acolhedor e confiável
- [ ] Frases curtas (máximo 2 linhas)
- [ ] Parágrafos curtos (máximo 4 linhas)
- [ ] Voz ativa preferencialmente
- [ ] Evitar dupla negação
- [ ] Usar "você" em vez de "o usuário"

### Precisão de Conteúdo
- [ ] NUNCA confundir marca com patente
- [ ] NUNCA confundir patente com desenho industrial
- [ ] NUNCA prometer resultado de registro
- [ ] NUNCA afirmar prazos como garantidos
- [ ] SEMPRE indicar que valores podem mudar
- [ ] SEMPRE referenciar "consulte o site oficial do INPI"
- [ ] SEMPRE separar claramente o que é orientação vs. o que é procedimento oficial
- [ ] Revisar conteúdo quando INPI atualizar regras ou sistemas

### Exemplos Pedagógicos Obrigatórios
Sempre que possível, usar exemplos concretos:
- **Marca**: "O nome 'Café Bom Dia' que identifica seu café especial"
- **Patente de Invenção**: "Um mecanismo novo que filtra água de um jeito nunca feito antes"
- **Modelo de Utilidade**: "Uma melhoria prática no cabo de uma panela que evita queimaduras"
- **Desenho Industrial**: "O formato exclusivo de uma garrafa ou o design de uma cadeira"
- **NÃO é PI**: "Uma receita de bolo, uma ideia de negócio, um método de ensino"

### Glossário Simplificado (manter atualizado)
| Termo técnico | Explicação simples |
|---|---|
| Propriedade Industrial | O conjunto de direitos sobre marcas, patentes e designs |
| INPI | O órgão do governo que cuida dos registros |
| GRU | A taxa que você paga ao governo para fazer o pedido |
| e-INPI | O cadastro online que você precisa ter no site do INPI |
| e-Marcas | O sistema online onde você faz o pedido de registro de marca |
| Classe Nice | A categoria do produto ou serviço que sua marca identifica |
| Busca de anterioridade | Verificar se alguém já registrou algo parecido |
| Reivindicações | A parte que diz exatamente o que você quer proteger na patente |
| Relatório descritivo | O documento que explica em detalhes como sua invenção funciona |
| Oposição | Quando alguém contesta seu pedido de registro |
| Exigência | Quando o INPI pede que você corrija ou complete algo no pedido |
| Concessão | Quando o INPI aprova seu registro |
| Anuidade | Taxa anual para manter a patente válida |
| Depósito | O ato de enviar oficialmente seu pedido ao INPI |
| Sigilo | Manter sua invenção em segredo antes de depositar a patente |

## 6. Responsabilidade sobre Integrações

### Dados Oficiais
- [ ] Sempre citar fonte: "Dados do INPI — consulte gov.br/inpi"
- [ ] Não garantir que dados estão atualizados em tempo real
- [ ] Informar data da última atualização quando possível

### Dados de Busca Não Oficial
- [ ] Disclaimer visível: "Esta busca é uma consulta auxiliar"
- [ ] Não usar linguagem que implique resultado definitivo
- [ ] Sempre oferecer link para busca oficial
- [ ] Nunca dizer "sua marca está livre" — dizer "não encontramos registros similares na nossa base, mas confirme no INPI"

### IA
- [ ] Toda saída de IA com disclaimer
- [ ] IA nunca afirma "você tem direito a..."
- [ ] IA sempre sugere "consulte um profissional" para casos complexos
- [ ] IA nunca inventa regras do INPI
